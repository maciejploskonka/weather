# WEATHER FORECAST APP

## General info

Check weather and forecast in every place!

## Technologies

- Vue.js
- HTML
- CSS/SCSS
- JavaScript
- Webpack

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn run serve
```

### Compiles and minifies for production

```bash
yarn run build
```

### Lints and fixes files

```bash
yarn run lint
```
